课程预备知识：

本课程紧密联系"数据库技术及应用（生物信息）"理论课程, 要求学生具有一定的计算机基础（计算机基础，计算机网络技术，操作系统基础，C/C++语言，JAVA以及GOLANG语言）, HTML标记语言以及Web相关知识基础, 并有一 定的编程能力。 同时, 本上机部分的实践活动需要在装有CENTOS系统的计算机上完成, 需要学生对Linux有一定的了解.

上机环境：

本课程将采用流行的Oracle 19.3(Linux版，对于Windows版，同学门可以自行练习) 1. CENTOS Linux 7.x; 2. Oracle 19.3C Linux版; 3. Oracle SQL Developer 23.1; 4. Oracle SQL Developer Modeler 23.3; 5. MyEclipse 2021 and Eclipse 2023; 6. GCC 8.x-9.x, Java 1.8-11, GOLAN 1.20+;

课程目的：

1. 掌据基本的Linux操作系统使用技巧, 包括网络配置，执行命令，文件传输等。 了解Linux与其他OS不同之处, 及其优势特点。 2. 通过实机练习, 让学生掌握基本的关系数据库技巧, 并加深理论课相关知识点, 特别是Oracle和MONGODB。

课程要求：

认真学习每个练习的基本操作，明了每个操作所对应的意义，特别是操作输出文件以及日志文件内容。要求学生详细记录相关操作结果，并对结果的正确与否进行分 析。要明 了输出数据的理论和实际的意义，要能对基本的问题有一定的解决能力。 上机练习结束后，要对整个上机过程形成一个完整的报告，按指定格式撰写，要有清晰的层次，分析问题时要有科学性和逻辑性。
